# MyBudget

Este repositório reúne o trabalho desenvolvido para o Case IEEE, seguindo as orientações do desafio e focando em entregar uma solução clara, funcional e bem organizada. A ideia é mostrar nossa capacidade de estruturar um projeto, colaborar em equipe e aplicar conceitos fundamentais de programação.

## Objetivo:

Desenvolver uma solução que atenda aos requisitos propostos, mantendo código limpo, boas práticas e uma documentação que facilite o entendimento do projeto.

## Tecnologias Utilizadas:

Linguagem: javaScript, HTML e CSS.

Ferramentas: Git, GitHub, MySQL e insomnia.

## Equipe

Projeto desenvolvido em conjunto pelos integrantes do grupo para o Case IEEE destinado ao ingresso na WolfByte - WEB.
